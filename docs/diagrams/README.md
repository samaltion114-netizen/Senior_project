# Nahd Backend Diagrams

This folder contains the core software diagrams for the full project.

## Included diagrams
- `01_erd.md` - Entity Relationship Diagram (database schema)
- `02_component_diagram.md` - Backend component diagram (apps/services/dependencies)
- `03_block_architecture.md` - High-level block architecture
- `04_deployment_diagram.md` - Deployment/runtime diagram (Docker services)
- `05_sequence_demo_flow.md` - End-to-end sequence diagram for the full demo scenario
- `06_state_diagrams.md` - State diagrams for Interview, Session, and Proof lifecycle
- `07_permissions_matrix.md` - Role/API access matrix

## Notes
- Diagrams are written in Mermaid for readability and easy editing.
- They match the current implementation in this repository.
